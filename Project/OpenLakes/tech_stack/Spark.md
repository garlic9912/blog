`Apache Spark` 是一个**开源的大规模数据处理引擎**，可以理解为专门为海量数据设计的"超级计算器"，解决了传统工具（如 `Excel` 或单机 `Python`）在面对 TB 甚至 PB 级数据时的速度和内存瓶颈问题。
其核心特点可用以下三个词概括：
- **统一**：使用一套 API 就能完成批处理、实时流处理、SQL 查询、机器学习等多种任务，无需学习和维护多套系统
- **快速**：基于内存计算，在处理复杂任务（如迭代算法）时，速度可比 `Hadoop MapReduce` 快 10 到 100 倍
- **可扩展**：从一台笔记本电脑到数千台服务器的集群，都可通过简单的配置进行伸缩，处理不同规模的数据

# Spark 的生态系统
Spark 本身是一个核心引擎，围绕它构建了一整套强大的工具库，类似一个"大数据操作系统"，可以应对各种不同的数据任务
- **Spark Core**：Spark 的基础，负责分布式任务调度、内存管理和故障恢复
- **Spark SQL**：用于处理结构化数据。可以使用标准 SQL 查询 Hive、Parquet 等数据源，或将其它任务的计算结果作为表进行处理
- **Spark Streaming / Structured Streaming**：用于实时数据处理。它采用"微批处理"的方式，将源源不断的实时数据切成小块，再用 Spark 引擎快速处理，适用于需要低延迟响应的场景，如实时风控和仪表盘监控
- `MLlib`：`Spark` 的机器学习库，提供了分类、回归、聚类等主流算法，并可以自动分布式扩展
- `GraphX`：用于图计算，可以用来分析社交网络或实体间的复杂关系
