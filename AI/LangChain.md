# 核心定位
`LangChain` 是一个用于**开发大语言模型应用**的开源框架。它的核心目标是解决一个痛点：**让大模型（大脑）能连接外部世界（数据+工具）。**
如果没有LangChain，你每次调用大模型都需要手动写代码处理API密钥、拼接提示词、管理多轮对话记忆、处理JSON解析……极其繁琐。LangChain把这些“脏活累活”全部封装成了标准化的“积木”

# 核心架构
- **Models（模型层）**：统一接入`OpenAI`, `Anthropic`、智谱、通义千问等几十种模型，切换模型只需改一行代码。
- **Prompts（提示词模板）**：把固定的提示词变成可填充变量的模板（比如“请帮我给{客户名}写一封关于{产品}的邮件”）。
- **Chains（链）**：把多个步骤串起来。比如：先搜索资料 -> 再总结内容 -> 最后翻译成英文，这一串动作就是一条“链”。
- **Retrievers（检索器）**：专门连接外部知识库（如PDF、网页、向量数据库），让AI能“开卷考试”，也就是做RAG（检索增强生成）。
- **Memory（记忆）**：自动帮AI记住之前的对话，解决“聊完就忘”的问题。
